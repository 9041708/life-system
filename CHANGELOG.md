## v2.14.0 (2026-07-12)

### L币系统
- **L币系统**：全新虚拟货币体系，支持余额、流水、套餐购买AI次数。L币可通过邀请注册、签到奖励、记账奖励、管理员发放等方式获取，支持小数精度（小数点后两位）
- **AI Token 计费**：AI 调用改为按 DeepSeek 返回的 actual total_tokens 计费，通过 L币自动扣除（向上取整）。管理端可配置 1 L币 = N Token 汇率
- **邀请注册**：邀请码生成、注册奖励（双方各获 L币）、落地页显示邀请人信息、阶梯式 L币 奖励配置
- **股市 L币 集成**：默认资金改为 0，进入股市需 L币兑换虚拟资金，支持反向赎回（含手续费）。管理端可配置汇率和手续费率
- **首页 L币卡片**：仪表盘增加 L币余额卡片
- **签到/记账 L币 奖励**：连续签到 N 天奖 L币（阶梯制），记账每笔奖 L币（日上限50笔），管理员可开关+配置金额
- **L币管理**：管理员专用选项卡，含套餐弹窗增删改、发放/扣除 L币弹窗（支持用户搜索+批量操作）、Token汇率、签到/记账奖励规则、股市兑换参数统一配置
- **性能优化**：启动阶段 DDL 迁移改为一次性标记检查，每次请求减少 ~42 条数据库查询；预算 N+1 改为批量聚合；PDO 启用持久连接
- **Bug 修复**：修复多处 PHP 语法错误（括号误写分号、正则未转义、BOM头）；论坛助手忽略作者过滤；股市兑换弹窗实时计算；settings 页面模块化保存

## v2.3.0 (2026-06-22)

### 我的人生 - 娱乐模块
- **性别选择**：游戏开始时可选择男生/女生，事件根据性别匹配（gender='all' 或匹配当前性别）
- **事件防重触发**：同一事件在同一局人生中不会重复触发，解决"老伴离世"同年重复出现的问题

- **快进10年**：快进按钮改为一次快进10年，中途可随时停止
- **成就殿堂**：新增独立成就页面，可查看所有成就及累计解锁次数
- **历史回顾增强**：每条历史记录下方显示该次人生获得的成就徽章
- **死亡选项机制**：选项支持 `die: true` 标记，选择后立即结束人生
- **数据库兼容**：life_records 表自动新增 gender 字段，旧数据默认 male

### Bug 修复
- 修复选择"安详离世"后继续活多年的问题（新增 `die` 标记判断）
- 修复 99 岁时属性值超出 TINYINT 范围导致 SQL 报错（写入前强制 cap）
- 修复"事件不存在"报错：`event_id=0` 的平凡事件不再触发数据库查询
- 修复"老伴离世重复触发"bug：事件触发后在 `life_log` 中记录并排除重选
- 修复 `apiStart()` INSERT 参数数量与占位符不匹配问题（SQLSTATE[HY093]）

### 数据补充
